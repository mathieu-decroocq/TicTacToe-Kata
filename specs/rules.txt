Rules :
    a game is over when all fields in a row are taken by a player
    players take turns taking fields until the game is over
    a game is over when all fields in a diagonal are taken by a player
    a game is over when all fields are taken
    there are two players in the game (X and O)
    a game has nine fields in a 3x3 grid
    a game is over when all fields in a column are taken by a player
    a player can take a field if not already taken

Regles :

    - une partie est terminée lorsque tous les champs d'une rangée sont occupés par un joueur
    - les joueurs jouent à tour de rôle jusqu'à la fin du match
    - une partie est terminée lorsque tous les champs d'une diagonale sont occupés par un joueur
    - une partie est terminée lorsque tous les champs sont occupés
    - il y a deux joueurs dans le jeu (X et O)
    - un jeu a neuf champs dans une grille 3x3
    - une partie est terminée lorsque tous les champs d'une colonne sont occupés par un joueur
    - un joueur peut prendre un champ s'il n'est pas déjà pris
